## Get started in four steps

Get the Examen App running locally and start creating exams for your students.

<Steps>
  <Step title="Clone the repository">
    Clone the Examen App repository to your local machine:

    ```bash
    git clone https://github.com/eric-vc/examen-app.git
    cd examen-app
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required npm packages:

    ```bash
    npm install
    ```

    The app uses the following key dependencies:
    - **React 19.2.0** - UI framework
    - **React Router DOM 7.13.0** - Client-side routing
    - **Firebase 12.8.0** - Backend and authentication
    - **Vite 7.2.4** - Build tool and dev server
  </Step>

  <Step title="Configure Firebase">
    Set up your Firebase configuration in `src/firebase.js`. Replace the configuration with your own Firebase project credentials:

    ```javascript src/firebase.js
    import { initializeApp } from "firebase/app";
    import { getFirestore } from "firebase/firestore";
    import { getAuth } from "firebase/auth";

    const firebaseConfig = {
      apiKey: "YOUR_API_KEY",
      authDomain: "your-project.firebaseapp.com",
      projectId: "your-project-id",
      storageBucket: "your-project.firebasestorage.app",
      messagingSenderId: "YOUR_SENDER_ID",
      appId: "YOUR_APP_ID"
    };

    const app = initializeApp(firebaseConfig);
    export const db = getFirestore(app);
    export const auth = getAuth(app);
    ```

    <Note>
      You'll need to create a Firebase project at [console.firebase.google.com](https://console.firebase.google.com) and enable Firestore Database and Authentication.
    </Note>
  </Step>

  <Step title="Start the development server">
    Run the Vite development server:

    ```bash
    npm run dev
    ```

    Your app will be available at `http://localhost:5173`. The dev server includes hot module replacement (HMR) for instant updates as you code.
  </Step>
</Steps>

## Verify your installation

Once the development server is running, you should see:

<CardGroup cols={2}>
  <Card title="Home Page" icon="house">
    The main page displays available exams loaded from Firebase. If no exams exist yet, you'll see "Cargando exámenes disponibles..."
  </Card>

  <Card title="Admin Panel" icon="user-gear">
    Access the teacher panel at `/admin` to create and manage exams.
  </Card>

  <Card title="Theme Switcher" icon="palette">
    Test the three built-in themes: Light (🌞), Dark (🌙), and Warm (☕).
  </Card>

  <Card title="Routing" icon="route">
    Navigate between pages using React Router. The app uses the basename `/examen-app` for GitHub Pages deployment.
  </Card>
</CardGroup>
