# Welcome to Examen App

Examen App is a comprehensive web-based platform designed for educational institutions to create, manage, and administer diagnostic exams. Built with React and Firebase, it provides a complete solution for both students taking exams and teachers managing assessments.

## Who Is This For?

<AccordionGroup>
  <Accordion title="Students">
    Students use Examen App to take diagnostic exams assigned by their teachers. The platform validates student credentials, randomizes questions, and provides instant feedback with detailed results.
  </Accordion>
  
  <Accordion title="Teachers">
    Teachers create and manage exams, build question banks, configure attempt limits, and monitor student performance. The admin panel provides complete control over all aspects of exam administration.
  </Accordion>
  
  <Accordion title="Educational Institutions">
    Schools and universities can deploy Examen App to standardize diagnostic testing across multiple academic units with centralized result tracking and reporting.
  </Accordion>
</AccordionGroup>

## Platform Highlights

### For Students
- **Simple Registration**: Enter your name, control number, and academic unit
- **Randomized Questions**: Each attempt presents questions in random order
- **Instant Results**: Get immediate feedback with detailed answer breakdown
- **PDF Export**: Download and print your results (when enabled by teacher)

### For Teachers
- **Exam Creation**: Define exam parameters including title, subject, and attempt limits
- **Question Management**: Add questions manually or import via CSV
- **Result Tracking**: View and filter results by student, exam, or academic unit
- **Bulk Operations**: Import/export questions and results in CSV format

### Technical Features
- **Firebase Authentication**: Secure teacher login with email/password
- **Firestore Database**: Real-time data synchronization for exams and results
- **React Router**: Smooth navigation between student and teacher interfaces
- **Responsive Design**: Works seamlessly on desktop and mobile devices
